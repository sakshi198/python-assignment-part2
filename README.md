# Python Assignment - Part 2: Restaurant Order Management System

This project implements a Restaurant Order Management System using Python dictionaries and lists.

## Tasks Covered

### Task 1: Menu Exploration
- Displayed menu by category
- Counted total and available items
- Found most expensive item
- Filtered items under ₹150

### Task 2: Inventory Monitoring
- Checked stock status
- Identified unavailable items
- Identified items to reorder

### Task 3: Order Processing
- Took user input for orders
- Validated availability and stock
- Calculated total bill
- Updated inventory

### Task 4: Sales Analysis
- Calculated total revenue
- Counted total orders
- Found best-selling item
